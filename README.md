# Virtual Desktop Strategy & Readiness Assessment

A browser-based assessment for evaluating Azure Virtual Desktop, Windows 365, and hybrid virtual desktop readiness.

The main artifact is [Virtual-Desktop-Strategy-Readiness-Assessment.html](./Virtual-Desktop-Strategy-Readiness-Assessment.html). It is a self-contained HTML application that runs locally in the browser with no backend or deployment setup.

## What It Does

- Covers Azure Virtual Desktop, Windows 365, and hybrid scenarios
- Guides a structured readiness and platform-fit conversation across business, persona, application, infrastructure, security, endpoint, and operations topics
- Converts inputs into recommendation signals, readiness indicators, and summary outputs
- Ships as a portable static web app that is easy to open, demo, and publish

- Multi-tab guided assessment with branching logic
- Live scoring across AVD, Windows 365, and hybrid patterns
- Progress tracking, section notes, and readiness indicators
- Visual outputs powered by Chart.js and a geographic view powered by Leaflet
- Single-file delivery model for portability and simple demos

GitHub Pages entry is [index.html](./index.html), which routes directly into the main assessment.
