# LinkedIn Post Copy

I turned a traditional VDI discovery questionnaire into an interactive product concept.

Over the last stretch of work, I designed and built **Virtual Desktop Strategy & Readiness Assessment**: a lightweight web-based decision-support experience for Azure Virtual Desktop, Windows 365, and hybrid desktop strategy.

The goal was simple: replace static intake sheets with something more useful.

Instead of capturing answers in a document, the experience now:
- guides the discovery conversation
- adapts based on migration and workload inputs
- surfaces readiness gaps and migration risks
- generates a structured decision brief with live platform-fit signals

Technically, it is a self-contained HTML product concept using Chart.js and Leaflet, designed to be easy to share, demo, and evolve.

What I like most about this project is the shift in mindset: taking a consultant workflow and treating it like a product problem.

That meant focusing not just on the questions, but on the experience, structure, signals, and outputs the user gets back.

If you work in end-user computing, Azure Virtual Desktop, Windows 365, or solution discovery, I would be interested in how you approach turning advisory work into reusable tools.
